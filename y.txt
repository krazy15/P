1. HADOOP WORDCOUNT

---

Objective:
To count frequency of words using Hadoop MapReduce.

Explanation:
MapReduce works in 3 phases:

* Map → breaks data into key-value pairs (word,1)
* Shuffle → groups same words
* Reduce → sums counts

Procedure:

# Start Hadoop services

start-dfs.sh
start-yarn.sh

# Create input file

nano input.txt

# Example text:

# big data is big data

# Create HDFS directory and upload file

hdfs dfs -mkdir /input
hdfs dfs -put input.txt /input

# Run WordCount program

hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-*.jar wordcount /input /output

# View result

hdfs dfs -cat /output/part-r-00000

Expected Output:
big 2
data 2
is 1

2. LOG ANALYSIS USING MAPREDUCE (SIMULATION)

---

Objective:
Analyze logs to find frequent IP addresses and error types.

Explanation:
We simulate MapReduce:

* Map → split logs
* Reduce → count frequency

Code:
from collections import defaultdict

logs = [
"192.168.1.1 ERROR",
"192.168.1.2 INFO",
"192.168.1.1 ERROR"
]

ip_count = defaultdict(int)
error_count = defaultdict(int)

for log in logs:
ip, status = log.split()
ip_count[ip] += 1
error_count[status] += 1

print("IP Frequency:", dict(ip_count))
print("Error Frequency:", dict(error_count))

7. SPAM CLASSIFICATION

---

Objective:
Classify emails as spam or not.

Explanation:

* Convert text → numbers
* Train classifier
* Predict

Code:
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score

texts = ["free money now","hello friend","win prize","meeting tomorrow"]
labels = [1,0,1,0]

vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

X_train,X_test,y_train,y_test = train_test_split(X,labels)

model = MultinomialNB()
model.fit(X_train,y_train)

pred = model.predict(X_test)
print("Accuracy:", accuracy_score(y_test,pred))

    logs = f.readlines()

    err = re.findall(r'\b[45]\d{2}\b', line)
    if ip: ips.append(ip[0])
    if err: errors.append(err[0])

print(Counter(errors).most_common(5))
