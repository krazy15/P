PRACTICAL 1: Linear Regression (Uber Fare Prediction)

THEORY:
Linear Regression is a supervised machine learning algorithm used to predict a continuous value. It establishes a linear relationship between dependent and independent variables.
Formula:
y = b0 + b1x1 + b2x2 + ... + bnxn
Where:
y = dependent variable (fare)
x = independent variables (features)
b = coefficients
R2 Score measures how well the model fits the data.
RMSE measures prediction error.

ALGORITHM:
1. Start
2. Load the Uber dataset
3. Display dataset information and summary statistics
4. Handle missing values by removing or imputing
5. Convert pickup datetime into useful features (e.g., hour)
6. Select relevant features for prediction
7. Detect and remove outliers using IQR method
8. Analyze correlation between features using heatmap
9. Split dataset into training and testing sets
10. Train Linear Regression model on training data
11. Predict fare using test data
12. Evaluate model using R2 Score and RMSE
13. Display results
14. Stop

CODE:
import pandas as pd
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt

from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score, mean_squared_error

# Load dataset
df = pd.read_csv("uber.csv")

# Preprocessing
df = df.dropna()
df['pickup_datetime'] = pd.to_datetime(df['pickup_datetime'])
df['hour'] = df['pickup_datetime'].dt.hour

# Feature selection
df = df[['fare_amount','pickup_longitude','pickup_latitude',
         'dropoff_longitude','dropoff_latitude','hour']]

# Remove outliers (IQR method)
Q1 = df.quantile(0.25)
Q3 = df.quantile(0.75)
IQR = Q3 - Q1

df = df[~((df < (Q1 - 1.5 * IQR)) | (df > (Q3 + 1.5 * IQR))).any(axis=1)]

# Correlation
sns.heatmap(df.corr(), annot=True)
plt.show()

# Split data
X = df.drop('fare_amount', axis=1)
y = df['fare_amount']

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train model
model = LinearRegression()
model.fit(X_train, y_train)

# Prediction
y_pred = model.predict(X_test)

# Evaluation
print("R2 Score:", r2_score(y_test, y_pred))
print("RMSE:", np.sqrt(mean_squared_error(y_test, y_pred)))

CONCLUSION:
The Linear Regression model successfully predicts Uber fare based on input features. Proper preprocessing, feature selection, and outlier removal improve the accuracy of the model. The model performance is evaluated using R2 Score and RMSE.

PRACTICAL 3: K-Nearest Neighbors (KNN) for Email Spam Detection

THEORY:
K-Nearest Neighbors (KNN) is a supervised machine learning algorithm used for classification. It classifies a data point based on the majority class among its k nearest neighbors.
Key Points:
- K determines number of neighbors
- Uses distance metric (usually Euclidean distance)
- Simple and effective for small datasets

ALGORITHM:
1. Start
2. Load the email dataset
3. Display dataset information
4. Handle missing values
5. Convert text data into numerical form using vectorization (e.g., TF-IDF)
6. Split dataset into training and testing sets
7. Train KNN model with chosen value of k
8. Predict class labels on test data
9. Compute Confusion Matrix
10. Calculate Accuracy, Precision, and Recall
11. Display results
12. Stop

CODE:
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import confusion_matrix, accuracy_score, precision_score, recall_score
from sklearn.feature_extraction.text import TfidfVectorizer

# Load dataset
df = pd.read_csv("spam.csv", encoding='latin-1')

# Select required columns
df = df[['v1','v2']]
df.columns = ['label','message']

# Convert labels to numeric
df['label'] = df['label'].map({'ham':0, 'spam':1})

# Text vectorization
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(df['message'])
y = df['label']

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train model
model = KNeighborsClassifier(n_neighbors=5)
model.fit(X_train, y_train)

# Prediction
y_pred = model.predict(X_test)

# Evaluation
print("Confusion Matrix:\n", confusion_matrix(y_test, y_pred))
print("Accuracy:", accuracy_score(y_test, y_pred))
print("Precision:", precision_score(y_test, y_pred))
print("Recall:", recall_score(y_test, y_pred))

CONCLUSION:
KNN successfully classifies emails as spam or not spam based on similarity with nearest neighbors. Model performance depends on the choice of k and quality of text features.

PRACTICAL 5: Artificial Neural Network (ANN) for Bank Customer Churn Prediction

THEORY:
Artificial Neural Network (ANN) is a supervised learning model inspired by the human brain. It consists of layers of neurons:
- Input Layer (features)
- Hidden Layers (processing)
- Output Layer (prediction)
Each neuron applies weights and activation functions to learn patterns.
Key Points:
- Handles complex and non-linear relationships
- Uses activation functions like ReLU and Sigmoid
- Learns through backpropagation

ALGORITHM:
1. Start
2. Load the customer dataset
3. Display dataset information
4. Handle missing values
5. Encode categorical variables (e.g., Gender, Geography)
6. Normalize/scale numerical data
7. Split dataset into training and testing sets
8. Build ANN model with input, hidden, and output layers
9. Train model on training data
10. Predict output on test data
11. Convert probabilities to binary output
12. Compute Confusion Matrix
13. Calculate Accuracy, Precision, and Recall
14. Display results
15. Stop

CODE:
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import LabelEncoder, StandardScaler
from sklearn.metrics import confusion_matrix, accuracy_score, precision_score, recall_score
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# Load dataset
df = pd.read_csv("churn.csv")

# Drop unnecessary columns
df = df.drop(['RowNumber','CustomerId','Surname'], axis=1)

# Encode categorical data
le = LabelEncoder()
df['Gender'] = le.fit_transform(df['Gender'])
df = pd.get_dummies(df, columns=['Geography'], drop_first=True)

# Split features and target
X = df.drop('Exited', axis=1)
y = df['Exited']

# Feature scaling
sc = StandardScaler()
X = sc.fit_transform(X)

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Build ANN
model = Sequential()
model.add(Dense(6, activation='relu', input_dim=X.shape[1]))
model.add(Dense(6, activation='relu'))
model.add(Dense(1, activation='sigmoid'))

# Compile model
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# Train model
model.fit(X_train, y_train, epochs=50, batch_size=32)

# Prediction
y_pred = model.predict(X_test)
y_pred = (y_pred > 0.5)

# Evaluation
print("Confusion Matrix:\n", confusion_matrix(y_test, y_pred))
print("Accuracy:", accuracy_score(y_test, y_pred))
print("Precision:", precision_score(y_test, y_pred))
print("Recall:", recall_score(y_test, y_pred))

CONCLUSION:
ANN successfully predicts customer churn by learning complex patterns in the dataset. Proper preprocessing and feature scaling improve model performance.

PRACTICAL 6: Gradient Descent to Find Local Minima

THEORY:
Gradient Descent is an optimization algorithm used to find the minimum of a function by iteratively moving in the direction of the negative gradient (slope).
For function:
y = (x + 3)^2
Derivative:
dy/dx = 2(x + 3)
Update Rule:
x = x - learning_rate * gradient
Key Points:
- Moves towards minimum step by step
- Learning rate controls step size
- Stops when gradient becomes small

ALGORITHM:
1. Start
2. Initialize x = 2
3. Set learning rate (e.g., 0.1)
4. Repeat for fixed number of iterations:
   a. Compute gradient = 2(x + 3)
   b. Update x = x - learning_rate * gradient
5. Display final value of x
6. Stop

CODE:
# Initialize value
x = 2
learning_rate = 0.1

# Iterations
for i in range(50):
    gradient = 2 * (x + 3)
    x = x - learning_rate * gradient

print("Value of x at minima:", x)

CONCLUSION:
Gradient Descent successfully finds the minimum value of the function. The algorithm converges to x = -3, where the function reaches its minimum.

PRACTICAL 8: K-Means Clustering on Sales Data

THEORY:
K-Means is an unsupervised learning algorithm used to group data into clusters based on similarity.
Working:
- Select number of clusters (k)
- Assign data points to nearest cluster centroid
- Update centroids
- Repeat until convergence
Elbow Method:
Used to find optimal k by plotting WCSS (Within Cluster Sum of Squares).
The "elbow point" indicates the best number of clusters.

ALGORITHM:
1. Start
2. Load the sales dataset
3. Display dataset information
4. Handle missing values
5. Select relevant features for clustering
6. Apply feature scaling (if needed)
7. Use Elbow Method to determine optimal k
8. Apply K-Means clustering with optimal k
9. Assign cluster labels to data
10. Visualize clusters (optional)
11. Display results
12. Stop

CODE:
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# Load dataset
df = pd.read_csv("sample.csv")

# Select features (example)
X = df.iloc[:, 1:3]

# Feature scaling
sc = StandardScaler()
X = sc.fit_transform(X)

# Elbow Method
wcss = []
for i in range(1, 11):
    kmeans = KMeans(n_clusters=i)
    kmeans.fit(X)
    wcss.append(kmeans.inertia_)

# Plot Elbow graph
plt.plot(range(1,11), wcss)
plt.xlabel("Number of Clusters")
plt.ylabel("WCSS")
plt.title("Elbow Method")
plt.show()

# Apply K-Means (example k=3)
kmeans = KMeans(n_clusters=3)
y_kmeans = kmeans.fit_predict(X)

# Add cluster labels
df['Cluster'] = y_kmeans

print(df.head())

CONCLUSION:
K-Means successfully groups data into clusters based on similarity. The Elbow Method helps determine the optimal number of clusters for better performance.

print("Accuracy:", accuracy_score(y_test, y_pred))
print("Precision:", precision_score(y_test, y_pred))
print("Recall:", recall_score(y_test, y_pred))

CONCLUSION:
The model successfully predicts passenger survival based on features like age, gender, and class. Proper preprocessing and feature selection improve classification performance.
